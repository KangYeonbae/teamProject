* 서비스(제품, 정보)를 제공하는 웹사이트 구현
- 주제: 자유 선정

* 준수사항
- 서비스는 하나의 주제로 정한다. (추상적인 상위 주제 X)
 예) 의류, 컴퓨터, 운동기구 => 개별주제는 OK
 예) 의류, 컴퓨터, 운동기구를 포함한 포털 X
- 프론트기술만 사용하여 구현한다. (OpenAPI 기술 X)
 + HTML, CSS, JavaScript, JQuery 필수
 + 수업시간에 배우지 않은 프론트기술 사용은 OK
   예: ChartJS(권장)
 + 백엔드 기술이거나 앞으로 배울 프론트 기술은 X
  (예: ChatGPT OpenAPI, 지도API, React등)
- 상담형 쳇봇 서비스를 구현한다.

* 평가기준
- 과제요구사항 준수여부
- 팀원간 협업 및 전체적인 스킬업
- Advantage
 + 디자인 완성도
 + 기술 완성도
 + 웹사이트에서 사용되는 초기 정형데이터 시뮬레이터로 생성
